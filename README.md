# 人生轨迹分析游戏

一个基于Web的互动游戏，帮助用户通过设置能力值、回答问题和做出选择来预测和探索人生轨迹。灵感来源于高考等关键节点，结合多种可能性，提供轻松愉快的用户体验。

## 功能特性

- **能力值设置**：智力、毅力、运气、社交、创造力（1-10级）
- **个性化问题**：兴趣、目标、团队偏好、家庭背景、冒险意愿
- **高考选择**：理科、文科、商科、体育、医学、法律等方向
- **大学后路径**：工作、读研、创业、留学、旅行等
- **动态预测**：基于输入生成个性化未来预测
- **简约UI**：卡片式布局，动画效果，进度条，反馈提示
- **响应式设计**：适配不同设备

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **样式**：自定义CSS，动画效果
- **交互**：原生JS，无框架依赖

## 如何使用

1. 克隆仓库：
   ```bash
   git clone https://github.com/qiuxiang2077/git_study.git
   cd git_study
   ```

2. 打开 `index.html` 在浏览器中运行：
   - 双击文件或使用本地服务器（如Live Server扩展）

3. 跟随步骤：
   - 设置能力值
   - 回答问题
   - 选择高考方向
   - 决定大学后路径
   - 查看预测结果
